# Kartaca Task

## **Technologies and techniques used in the project**

### **Frontend**

- **Next.js - Typescript**
- **Tailwind**
- **Socket.io-Client**
- **Axios**
- **Context API**
- **Toastify**
- **Yup**

### **Backend**

- **Node.js - Typescript**
- **Express.js**
- **Redis**
- **Socket.io**
- **Mongoose**
- **Bcrypt**

### **Database**

- **Mongo DB**

## **Run Locally**

Clone the project

```bash
  git clone https://github.com/mustafahincal/kartaca-task.git
```

Go to the project directory

```bash
  cd kartaca-task
```

Docker

```bash
  docker-compose up
```
